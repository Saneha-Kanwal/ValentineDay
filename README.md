# Valentine ❤️ - Romantic Interactive Web App

A modern, romantic, multi-screen interactive web application with smooth animations and transitions.

## Features

- **4 Interactive Screens** with smooth transitions
- **Floating Heart Particles** background animation
- **Dark Theme** with soft pink and rose gradient glow
- **Fully Responsive** mobile-first design
- **Smooth Animations** using Framer Motion
- **Elegant Typography** with romantic fonts

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- Framer Motion

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

## Screen Flow

1. **Screen 1**: Sequential animation with character, heading, and button
2. **Screen 2**: Cuteness measuring with animated counter and progress bar
3. **Screen 3**: Clickable hearts that reveal sweet messages
4. **Screen 4**: Romantic note with elegant text display

Enjoy! ❤️
