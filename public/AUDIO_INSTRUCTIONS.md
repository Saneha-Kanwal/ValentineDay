# Audio File Instructions

## Adding Your Romantic Song

1. Place your audio file in this `public` folder
2. Name it: `romantic-song.mp3` (or `romantic-song.ogg`)
3. Supported formats: MP3, OGG

## Recommended Song
- "Tu He Ha Mery Dil Ke Tamana" 
- "Mahe Mahe"

## File Location
The audio file should be placed at:
```
public/romantic-song.mp3
```

## How It Works
- The song will automatically start playing when the website loads
- It will loop continuously in the background
- Users can control volume and play/pause using the controls in the bottom-right corner
- The controls auto-hide after 3 seconds but can be shown again by clicking the music icon

## Browser Autoplay Policy
Note: Some browsers may block autoplay. If the song doesn't start automatically:
- Click the play button that appears in the bottom-right corner
- The song will then play continuously
