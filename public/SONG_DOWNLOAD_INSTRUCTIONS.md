# Song Download Instructions

## Song Required
**"Vy Hania, Vy Dil Jania"** - At least 3 minutes long

## How to Download and Add the Song

### Method 1: Using YouTube Downloader (Recommended)
1. Search for "Vy Hania Vy Dil Jania" on YouTube
2. Copy the YouTube video URL
3. Use a YouTube to MP3 converter website like:
   - y2mate.com
   - ytmp3.cc
   - onlinevideoconverter.com
4. Download as MP3 format
5. Make sure the file is at least 3 minutes long

### Method 2: Using Music Streaming Services
1. If you have the song on Spotify, Apple Music, or other platforms
2. Use a music downloader tool (make sure it's legal in your region)
3. Download as MP3 format

### Method 3: If You Have the File Already
1. Make sure it's in MP3 format
2. If it's shorter than 3 minutes, you can:
   - Use audio editing software to loop it
   - Or find a longer version online

## After Downloading

1. **Rename the file** to: `romantic-song.mp3`

## File Requirements

## Verify It Works
1. Start your dev server: `npm run dev`
2. Open the website
3. The song should start playing automatically
4. Check the audio controls in the bottom-right corner

## Troubleshooting
- If the song doesn't play, check the browser console for errors
- Make sure the file name is exactly `romantic-song.mp3`
- Ensure the file is in the `public` folder, not `src` folder
- Check that the file is a valid MP3 format
